# Grantify AI - Visioners

## Problem Statement
AI enabled student loan eligibility, loan application, disbursal and agentic scholarship approval system.

## Key Features
- **AI-Powered Loan Eligibility Analysis**: Intelligent assessment of student profiles to determine loan suitability.
- **Automated Document Verification**: Streamlined verification process for student documents using AI.
- **Agentic Scholarship Approval System**: An intelligent system for reviewing and approving scholarship applications.
- **Real-time Disbursement Tracking**: Monitor the status of loan and scholarship disbursements in real-time.
- **Admin & Bank Panels**: Dedicated interfaces for scholarship authorities and banking institutions to manage workflows.
- **Student Dashboard**: A comprehensive portal for students to apply for schemes and track their status.
- **Secure Document Management**: Encrypted and organized storage for sensitive student information.
- **Communication Portal**: Integrated channel for seamless interaction between students and financial institutions.

## Technology Stack
- **Frontend**: React.js with Tailwind CSS
- **Backend**: Node.js & Express
- **Database**: MongoDB (Local)
- **AI Integration**: Custom AI models for eligibility and document analysis

## Getting Started
1. Install dependencies: `npm install`
2. Start the development server: `npm run dev`
3. Launch the backend server: `node server/server.js`
